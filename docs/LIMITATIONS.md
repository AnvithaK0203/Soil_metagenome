# Limitations

- No directly observed crop or suitability target exists.
- Agronomic metadata remains incomplete.
- Soil pH, NPK, moisture, and temperature are not yet available as a complete sample-level table.
- The dataset contains only `24` samples.
- The microbial feature space remains high-dimensional even after filtering.
- The current SMHI is a descriptive prototype and is not a validated soil-health diagnosis.
- The repository does not currently support a defensible supervised crop-prediction claim.
