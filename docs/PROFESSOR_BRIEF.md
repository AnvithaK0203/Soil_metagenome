# Professor Brief

This repository now reflects the current real state of the capstone project.

## What has been completed

- recovery and validation of the intended `24` SRR runs
- reconstruction of a sample-level merged microbiome dataset
- exploratory microbiome profiling
- diversity and composition analysis
- ordination, clustering, and similarity analysis
- reduced feature-table construction
- a transparent prototype Soil Microbiome Health Index in Cluster 1

## What has not been completed

- supervised crop prediction
- crop recommendation
- any externally validated agronomic scoring system

## Why supervised modeling is blocked

- no directly observed crop/suitability label is present
- agronomic metadata are incomplete
- the current sample size is too small for strong high-dimensional predictive claims

## Current correct interpretation

The project is presently a validated exploratory soil microbiome analysis workflow with a prototype internal SMHI, plus a future supervised-workflow scaffold waiting for real metadata and labels.
