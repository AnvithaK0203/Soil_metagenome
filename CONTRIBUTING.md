# Contributing

## Scope

This repository should remain scientifically conservative. Do not introduce files or claims that imply validated crop prediction unless directly observed targets and agronomic metadata have been added.

## Preferred contribution pattern

1. Preserve provenance.
2. Keep raw heavy sequencing artifacts out of Git.
3. Prefer reproducible scripts over one-off edits.
4. Update the relevant docs when outputs or scientific status change.
5. Keep Cluster 1 and Cluster 2 separated.

## Before opening a change

- confirm whether the change belongs in Cluster 1 or Cluster 2
- avoid adding fabricated labels or inferred agronomic targets
- document new assumptions and limitations
- keep reviewer-facing files concise
