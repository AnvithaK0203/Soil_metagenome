# Limitations

- The dataset contains only `24` samples.
- The feature space is still high-dimensional even after filtering.
- No directly observed crop label, suitability label, or measured agronomic target is present.
- Soil chemistry variables such as pH, NPK, moisture, and temperature are not available at sample level in a complete form.
- The prototype health index is descriptive only and is not externally validated.
- Any crop recommendation or crop suitability claim would currently be unsupported.
