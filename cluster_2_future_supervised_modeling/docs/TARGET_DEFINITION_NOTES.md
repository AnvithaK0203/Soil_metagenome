# Target Definition Notes

Acceptable targets must be directly observed or externally curated.

## Acceptable examples

- crop actually grown for the sampled field or plot
- measured crop outcome after the sampling event
- externally curated crop-suitability label with a documented protocol
- externally measured soil-health score from a real assay

## Unacceptable examples

- target inferred from country, state, or district
- target inferred from SRR/BioSample accession
- target inferred from PCA coordinates or unsupervised cluster membership
- target inferred from dominant phylum or diversity alone

## Minimum checks before use

- observation timing is clear
- one target value per sample
- enough samples per class
- no leakage from post-outcome variables
