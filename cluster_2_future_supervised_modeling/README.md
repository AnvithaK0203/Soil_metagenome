# Cluster 2: Future Supervised Modeling

This cluster is a blocked future workflow.

It exists to prepare the repository for later supervised analysis once real sample-level metadata and directly observed targets are available. It is not executable in the current repository state.

## Required future inputs

- soil pH
- nitrogen / phosphorus / potassium
- moisture
- temperature
- observed crop outcome, crop grown, or an externally defined crop-suitability label

## Guardrail

No labels or metadata are fabricated here. Blank templates are provided so future data can be integrated without mixing it into the current unsupervised workflow.
